# 🎓 Student Performance Dashboard

## 📌 Project Overview

This project is an interactive Student Performance Dashboard built using Python, Pandas, and Streamlit. The dashboard helps analyze student exam performance and provides insights through visualizations and statistics.

## 🚀 Features

* View student performance data
* Analyze Math, Reading, and Writing scores
* Gender-wise performance comparison
* Test preparation course analysis
* Parental education impact analysis
* Interactive charts and visualizations
* Summary statistics and insights

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Streamlit
* Matplotlib
* Seaborn

## 📂 Dataset

The dataset contains information about students, including:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

## 📊 Dashboard Visualizations

* Score Distribution Charts
* Gender-wise Analysis
* Correlation Heatmap
* Average Score Comparison
* Test Preparation Impact Analysis

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-performance-dashboard.git
```

2. Navigate to the project folder

```bash
cd student-performance-dashboard
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the Streamlit app

```bash
streamlit run app.py
```
